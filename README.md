# ClearJP - Proyecto Frontend (2025)

## Estructura del proyecto
```
ClearJP/
├── index.html
├── servicios.html
├── productos.html
├── faq.html
├── login.html
├── registro.html
├── css/
│   └── style.css
├── js/
│   ├── main.js
│   └── auth.js
├── assets/
│   ├── 500x500_Logo clearjp.png    # pega aquí tu logo (obligatorio)
│   └── placeholder-product.png
└── docs/
    └── style-guide.html
```

## Cómo ejecutar en Visual Studio Code
1. Abrir la carpeta `ClearJP` en VS Code.
2. Instalar la extensión **Live Server** (Ritwick Dey).
3. Abrir `index.html` y seleccionar **Open with Live Server** para ver el sitio en el navegador y recargar en caliente.

## Notas importantes
- Sustituye `assets/500x500_Logo clearjp.png` por tu archivo de logo real (500x500 PNG, fondo transparente o círculo degradado).
- El proyecto es estático (HTML/CSS/JS). Para funcionalidades reales de login y reservas necesitarás implementar un backend o usar servicios externos.
- Los scripts `js/auth.js` y `js/main.js` implementan validación de ejemplo en cliente, pensada para demo.
